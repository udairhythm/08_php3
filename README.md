# 07_php2 ブックマークアプリ

## 紹介と使い方

  - Google Books APIsから検索して、お気に入りをMysqlに検索結果を保存

  - 本の名前とURL, コメントを記載し保存可能

## 工夫した点

  - 本の名前とURL, コメントを記載し保存可能
  - 検索ボックス追加
  - 見た目調整

## 苦戦した点

  - お気に入り(Bookmark)を検索された本の横にボタンを設けるなどして実装したいが未完成
  - Google Books APIだと本の種類が少ない。Amazonでやりたい
  - ブックマーク一覧からリンクで本に飛べるようにしたい
